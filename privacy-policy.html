UI Layout Preview v1.0.0 — Fitnes maslahat ilovasi maketi

Maxfiylik siyosati

La1 foydalanuvchilarning maxfiyligini hurmat qiladi. Ushbu siyosat hozirgi UI maket (layout preview) versiyasiga tatbiq etiladi.

1. Ilova haqida

La1 — fitnes va faol hayot bo'yicha maslahat beradigan ilova interfeys maketidir. Bu versiya faqat dizayn namunasi sifatida taqdim etiladi. Tibbiy yoki professional fitnes xizmati emas.

2. Yig'iladigan ma'lumotlar (hozirgi maket versiya)

Hozirgi UI maket versiyasida foydalanuvchi ma'lumotlari serverga yuborilmaydi va uzoq muddat saqlanmaydi.

  • Profil sozlash ekranida kiritilgan ma'lumotlar faqat qurilmada ko'rsatish uchun ishlatiladi (maket)
  • Profil rasmi faqat mahalliy tanlash/kamera orqali ko'rsatiladi, serverga yuklanmaydi
  • SMS tasdiqlash haqiqiy emas — hech qanday telefon raqam yuborilmaydi
  • Fitnes statistikalar va AI chat xabarlari namuna (mock) ma'lumotlardir

3. Ruxsatlar

  • Kamera — profil rasmi UI maketini ko'rsatish uchun (ixtiyoriy)
  • Galereya — profil rasmini tanlash uchun (ixtiyoriy)

Boshqa ruxsatlar (joylashuv, bildirishnomalar va hokazo) hozirgi maket versiyada so'ralmaydi.

4. Uchinchi tomon xizmatlari

UI maket versiyasida ma'lumotlar uchinchi tomon API, Firebase yoki analytics xizmatlariga yuborilmaydi.

5. Bolalar

Ilova 13 yoshdan kichik bolalar uchun mo'ljallanmagan.

6. O'zgarishlar

To'liq versiya chiqarilganda ushbu siyosat yangilanadi. Maket versiya yangilanishlari ushbu sahifada e'lon qilinadi.

7. Aloqa

Maxfiylik bo'yicha savollar uchun: support@la1.app
